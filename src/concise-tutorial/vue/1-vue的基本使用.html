1. 通过 script 标签引入的形式体验 vue ​

对于前端初学者来说，如果 node 基础不扎实，可能会导致对基于脚手架搭建的 vue 项目感到畏惧，所以我们先从简单的 html 开始说起。

首先新建一个 html 文件，并且通过 script 的形式引入 vue.js

html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
    <!-- 截止2024.07.16是这个cdn地址，如有偏差请以vue官网为准: https://cn.vuejs.org/ -->
    <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
  </head>
  <body></body>
</html>
